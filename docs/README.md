# Lumina Web3 - Marketing Site

## Overview
Marketing website for Lumina Web3 crypto marketing agency.

## Tech Stack
- Next.js 16
- React 19
- Tailwind CSS
- Deployed on Vercel

## Local Development
```bash
npm install
npm run dev
```

## Environment Variables
Copy `env.example` to `.env.local` and fill in values.

## SEO
- sitemap.xml and robots.txt in /public
- OG tags configured in app/layout.tsx
- Structured data (JSON-LD) in index.html
